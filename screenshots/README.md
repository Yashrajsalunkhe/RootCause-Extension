# Screenshots Directory

This directory contains screenshots used in the main README.md file.

## Required Screenshots

Please add the following screenshots to showcase the extension:

### 1. `analysis-panel.png`
- **What to capture:** Full RootCause DevTools panel showing analysis results
- **Recommended size:** 1280x800px or similar
- **Content:** Should show the complete panel with performance issues listed

### 2. `metrics-overview.png`
- **What to capture:** Top section showing performance metrics
- **Content:** Performance grade, page size, load time, number of requests
- **Recommended size:** 800x400px or similar

### 3. `root-cause-details.png`
- **What to capture:** Detailed view of a specific performance issue
- **Content:** Shows the explanation and actionable recommendations
- **Recommended size:** 800x600px or similar

## How to Take Screenshots

1. Load the extension in Chrome
2. Open DevTools (F12) on any website
3. Navigate to the "RootCause" tab
4. Click "Analyze Page"
5. Take screenshots of the results panel

## Tips for Good Screenshots

- Use a clean, well-lit theme (dark mode works well)
- Ensure text is readable
- Capture meaningful performance data (use a real website like news sites or e-commerce)
- Crop unnecessary browser chrome/UI when possible
- Save as PNG for better quality with UI elements

## Image Format

- **Format:** PNG (preferred) or JPG
- **Compression:** Use moderate compression to keep file size reasonable
- **Max file size:** Keep under 500KB per image if possible
